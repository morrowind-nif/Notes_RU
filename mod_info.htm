Mod Info

Утилита для генерации отчетов и просто вывода подробных сведений о плагине.
Опции редактирования - отсутствуют.
 
На самом деле архиполезная вещь, как для быстрого осмотра плагина и получения подробной информации о кол-ве объектов им добавляемых, так и для работы с переводами, или иными правками!
 
Поскольку позволяет экспортировать информацию из плагина в текстовые файлы.
Т.е. получить единственный файл со всеми скриптами - возможно!
(такой файл крайне удобен для пакетного обновления записей, т.е. заменить что-то на "что-то иное" парой кликов.)
Опции Импорта в самой программе - не предусмотрено.
 
Примечание.
Для этого существуют Mod Text Merger и Mod Text Importer.
Которые замечательно работают с файлами экспортированными из Mod Info.
Остается лишь удивляться, что автор не соединил все три программы в один пакет.
Да, это все три утилиты от одного автора.
 
Импорт записей в отдельные файлы МЕГА ПОЛЕЗЕН для сравнения двух версий одного плагина!
Т.е. есть версия 1.0 и есть версия 1.001 - в чем разница, либо забыто, либо вовсе не известно.
- Используя МодИнфо экспортируем записи, например скриптов, из первого плагина.
- затем из второго.
- полученные текстовые файлы загружаем (например) в ТОТАЛ КОМАНДЕР и через опцию Compare (сравнить) сравниваем!
- теперь остается просмотреть файл до следующей красной отметки, а при необходимости внести правки.
- исправленный файл, можно загрузить в Mod Text Importer, если нужно переклеить одни исправления, не меняя других.
 
Поистине, этот пакет наипервешее средство в руках переводчика плагинов!
Поскольку позволяет пакетно переводить тексты, просто бросая куски из текстовых файлов в онлайн переводчик и обратно в файл.
В обычном случае, пришлось бы сидеть в ТЕС КС, поштучно просматривать скрипты и выцеплять Сообщения в общей массе текста на глазок...
