Базовый пакет программ

Общие примечания о базовом пакете программ потребных для «продвинутого» модинга, т.е. того, что нужно для создания и обработки моделей под ТЕС3 МВ.

Примечание!
Для использование нескольких версий нифскопа, рекомендуется сделать Портайбл сборку оного!
Это позволит сохранить реестр чистым, а также избавит от конфликтов между версиями.
INI файл, к сожалению, появился только в 2.0 версии.
Все ранние версии используют реестр вындоуз для хранения ряда настроек.
4 . Гимп (Gimp), Фотошоп (Photoshop), ПайнтНет (PaintNet) – Инструменты для создания текстур для моделей.
Могут использоваться все вместе.
Важное требование к редактору, он должен уметь работать с ДДС файлами.
ДДС, не только экономит место и поддерживает альфа канал, но и позволяет создавать дополнительные эффекты.
5. ХинВию (XnView) – или иной браузерный просмотровщик 2д графики.
Просмотр текстур в виде превьюшек совершенно необходим для удобной работы и поиска текстур.
Главное достоинство ХинВию - просмотр ДДС файлов и еще множества иных форматов, которые можно пересохранить в любой иной формат.
К программам необходимы следующие плагины:
- Нифплагины к МАХу – существуют в нескольких версиях. См. раздел о плагинах к МАХ.
- Нифплагины к Блендеру. См. ссылки выше.
- ДДС плагины к 2д Редакторам.
Также ДДС плагин может понадобиться и для МАХ 3.1, 4.2.
Под старые версии Фотошопа, можно найти на старых сайтах по ТЕСу, например TES.AG.ru.
Также на MMH (моровинд моддинги хистори).
Под новые версии ФЩ - на сайте Nvidia.com
- Детриангулятор. Плагин для 3д МАХ, позволяет оптимизировать модель после ее Импорта.
 
Это был, так сказать, базовый джентльменский набор для модинга под МВ, который рекомендуется дополнить рядом вспомогательных программ и утилит.
1. ТоталКомандер (Total Commander) – для удобного копирования файлов.
И в целом, для удобства работы, например - смены дат у плагинов. Поддерживает подключение модулей.
В т.ч. для просмотра ДДС и БСА архивов.
 
2. 7зип (7zip) – архиватор c высочайшей степенью сжатия файлов.
 
3. Рар (WinRar) – то же самое, но с функцией добавления информации для восстановления, что бывает полезно.
 
4. ТесФайлз (TESFILES) или ТесМи (TesME) – утилиты, для извлечения состава плагинов из общей массы материала игры.
Т.е. это упаковщики избранных плагина.
 
5. Тезама (TES Advansed Mod Editor) – утилита, для быстрой очистки плагина, что бывает очень полезно!
 
6. БсаКомандер, или консольный БсаПакер (bsaPack) — это отдельные утилиты для работы с БСА архивами игры.
Существует некоторое кол-во этих программ.
 
7. Древняя версия ACDSee (2.4 к примеру, если повезет найти сами знаете где(С) - для быстрого просмотра картинок.
Обладая нулевым обвесом имеет превосходную скорость загрузки файлов.
Что может быть весьма полезным, для вспомогательных действий. Например менеджмента скриншотов.
Более новые версии, лучше обходить за сто верст лесом.
Из минусов, не умеет открывать ДДС файлы.
 
8. ScreenShot Creator - весьма полезная программка позволяющая делать скриншоты с авто сохранением.
Имеет массу полезных настроек. Легка в обращении и не подвязана на систему.
Незаменима при создании большого кол-ва иконок!
Т.к. позволяет заготовить скрины, сразу для всех объектов. Остается лишь создать экшен и batch в фотошопе - пара минут и несколько сотен новых иконок готовы. Жаль, что КС не умеет создавать подобные "скрипты" по обработке своих объектов...
 
И некоторые вспомогательные инструменты для создания моделей и текстур:
 
– FaceGen — программка умеет делать головы, с заготовками анимаций, для существ и нпс.
– ParticleIllusion — бывает полезна для создания разного рода анимированных текстур.
– СAT  —  плагин к 3д максу, может оказать весьма полезным при создании некоторых анимаций.
– SimCloth — также плагин к максу. Оказывается весьма полезным для создания анимаций ткани и физики твердых тел.
– DxRiper — небольшая программка позволяющая создавать интересные вещи.
Впрочем, по "обновленным" данным, MWSE 2.Х тоже умеет такое делать при этом лучше и быстрее!
 
Пожалуй, этого будет достаточно для "базового" модинга.
