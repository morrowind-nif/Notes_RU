TSEXP Animation

TesExporter
TesExporter экспорт
Animation
Use App Time - (Usually on) Uses time as a global value when on or a value relative the the animation start time when off.
Настройка экспорта анимаций.
Подробнее см в ФФЕкпортере.
 
По видимости, в ТЕСэкспортере, полностью заблокирован режим экспорта APP_INT.
 
Include in NIF
Include animation keyframe data in the regular .nif file (Usually on).
The animation data might likely end up in a seperate .kf file with an associated .nif file. 
These 2 files will be named as the original with an "X" prepended.
Если установлено, анимация будет записана только в Ниф файл, если нет, будут созданы Хниф и КФ файлы.
Аналог экспорта Nif with Animation Нифтулз модуля.
 
Это режим используется для экспорта простых постоянных анимаций.
Например, вращающихся механизмов в двемерских руинах.
И прочих подобных объектов, которым не будет создаваться дополнительное управление.
Как правило, всегда требует создание анимационной ноды.
 
Extract to .KF
Extract animation data to a seperate .KF file (Almost never used since .kf files are created automatically).
Аналог экспорта single kf with nif Нифтулз модуля.
 
Экспорт анимации в отдельный КФ файл.
Используется при создании существ и активаторов, которыми можно управлять через скрипты.
 
Biped Sample Rate
Biped Sample Rate - Since biped animation data must be sampled this is the frame sample ratio.
1:1 = Every frame (All player animations and some NPCs)
1:2 = Every 2 frames (Most NPCs and some creatures)
1:3 = Every 3 frames.
1:4 = Every 4 frames (Most creatures)
Относится к частоте кадров проигрывания анимации существами и объектами.
Влияет на скорость проигрывания оной в игре 1.1, 1.2, 1.3, 1.4 относительно заданной  в МАХе.
Как кажется, также сокращает размер файлов анимаций, за счет сокращения кол-во фреймов.
К сожалению, используется крайне редко т.к. нормально работает только в 4ом МАХ.
 
Biped Sample Rate – 1.1, 1.2, 1.3, 1.4 также будет влиять на размер модели в игре.
