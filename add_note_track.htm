Add Note Track

Дорожка текстовых заметок необходима для создании анимации существ, ряда активаторов и голов.
 
Как добавить Ноте Трек.
–     Выделить корневой объект сцены.
Для Голов - выделить основною модель головы.
Для существ, обычно, это корневая кость скелета Bip01, или Root Bone.
–     ПКМ —> DOPE SHEET —> NOTE TRACK —> ADD.
–     Выбрать инструмент установки ключей и расставить оные по треку.
–     ПКМ, по NOTE — позволяют вносить правки.
*если выделение с кости слетело, "заметки" пустеют! Потребуется заново выделить кость и ключ заметок.
*если открыто несколько Нот, будет показано одно значение во всех.
 
Примечание.
Чем больше ключей, тем медленней работает Track (в режиме Dope Sheet)!
Особенно если включены режимы Modify Subtree и Modify Child Keys!
В ряде случаев выделение ВСЕХ ключей может повесить 3д макс на ВЕСЬМА длительное время.
 
По мере возможности держите: Modify Child Keys отключенным!
Хотя это и может усложнять анимирование ряда объектов.
Включен  Modify Subtree.
Выключен Modify Child Keys.
