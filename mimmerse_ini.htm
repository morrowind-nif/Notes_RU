mimmerse.ini

Файл настроек ТЕС и НетИммерсе экспортеров.
Т.е. все версии NetImmerse\Gamebryo плагинов, в той, или иной, степени используют этот файл.
 
Файл создается ТЕСэкспортером в момент первого экспорта модели в МВ.
В нем сохраняются наборы установленных флагов и некоторых иные параметры по умолчанию.
 
То, что создает ТЕСэкспортер:

Special Purpose Gamebryo 3ds max Plug-in INI Options
To provide special capabilities in some cases and backwards compatibility in others, there are a few settings in your Gamebryo 3ds max Plug-in INI file which you may want to edit. This file is located at ${3dsmax}/plugcfg/mimmerse.ini. To use these options, use a text editor to add the line to the INI file.
Option
Description
UseAreaSuperSpray=1
Will cause SuperSpray to emit from an area based on the emitter icon size, instead of from a point [default: UseAreaSuperSpray=0].
UseOldSpinTime=1
For backwards compatibility of pre-NetImmerse 4.2.1 particle system "spin time" interpretation [default UseOldSpinTime=0].
UseOldBehavior=1
Double-sided materials now use a stencil buffer property to disable backface culling. Previous to NetImmerse 4.2 and Gamebryo 1.0, the geometry was doubled for the objects to which this material was assigned. The only side effect is that lighting will only occur on the forward facing side of the object, i.e. the side with normals.
As of NetImmerse 4.2 and Gamebryo 1.0, keyframe extraction sequences can be specified for more than one object at a time. Please review the Sequence Extraction documentation for more information and to find out about the greater flexibility this technique gives the artist.
This option will enable the old behavior for both of these changes.
