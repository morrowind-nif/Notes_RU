TES CS Hot Keys

Список известных горячих клавиш которыми можно пользоваться в КС.
LMB - левая кнопка мыши.
RMB - правая кнопка мыши.
MMB - средняя, ака зажатый ролик мыши.
ARR - клавиши стрелок на клавиатуре.
 
Если что-то пропущено, см. английскую версию.
GEneraL*
F1
справка.
F2
на выделенном, в меню, объекте - включает переименование его ИДЕ!
*либо медленно 2ЛКМ.
F3
поиск по заданной маске.
Т.е. сперва по Ктрл+Ф задается, что искать, затем ф3 для продолжения поиска
F4
показывает радиусы колиженов на моделях.
F5
перерисовать сцену, ака обновить
F10
Вызывает SceneGraph - здесь выводиться информация о сцене.
Можно смотреть свойства объектов.
 
Полезно для работы с ниф файлами!
 
 
СTRL + Z
отмена
CTRL + Y
redo. Возврат действия в обратную сторону.
СTRL + S
сохранить плагин
СTRL + F
поиск
СTRL + I
выводит информацию о Референсах
СTRL + Shift + tab
СTRL + tab
переход между вкладками объектов.
назад, вперед.
СTRL + Shift + V
Paste in Place
вставляет скопированный объект, на место копируемого.
хотя, по идее, должно вставлять в позицию любого другого выделенного объекта.
CTRL+SHIFT+ALT+I
Get data about references? Выводит информацию о референсах.
CTRL+SHIFT+R+MMB
Выдает еще одно окно о референсдата выделенного объекта.
СTRL + С
копировать.
СTRL + V
вставить
СTRL + D
Duplicate.
клонировать выделенный объект.
очень полезная опция!  Поскольку объект не меняет исходных координат.
СTRL + X
вырезать.
СTRL + ARR
прокручивание списков без снятия выделения с объекта.
СTRL + BackSpace
отмена действия.
СTRL + F10
сменяет курсор не снимая выделения с объекта.
Т.е. перекрестье сменяется на обычный курсор.
Возможно выполняет и иные функции.
 
 
A
освещение, вкл\выкл.
C
центрироваться на выбранный объект. Весьма ходовая кнопка при размещении объектов в мире.
D
снять выделение с выделенных объектов. Полезно при позиционировании камеры.
H
редактор ландшафта. Актуально только в экстерьерах.
F
уронить выделенные объекты.
L
показать источники света и их радиусы.
M
показать маркеры.  Источники звуков, маркеры дверей, источники света и пр.
T
вид сверху на выбранный объект.
W
варфрейм режим отображения. Показывает сетку моделей без текстуры.
Space
равно зажатому ролику мыши. Т.е. влияет на перемещение камеры.
ScrollLock
Включение переключения. Т.е. включить «ScrollLock» и можно клавишей PAUSE менять группы анимаций. Затем жать PrintScrin – существо, нпс начнут «дергаться» означенной анимацией.
PrintScrin
проигрывает пару фреймов анимации в окне рендера.
Включая звук и движения у существ. Проигрываются обычные идолы.
Т.е. можно увидеть некоторую анимацию НПС и существ.
Одно нажатие = один фрейм.
Звук проигрывается целиком, т.е. до конца файла.
Звук идет от всех объектов в сцене! Т.е. можно слышать амбиентный шум ячейки.
Pause
переключение групп анимации у существ в окне рендера.
При этом анимация проигрывается урезано! Руки не участвуют.
Работает только при включенном ScrollLock.
PgUp
в Preview Window приближение объекта.
PgDn
в Preview Window  удаление объекта.
PgUp
в Render Window - вращение камеры.
Home
в Render Window - вращение камеры в другую сторону.
Home и PgUp - в окне рендера крутят камеру по\против часовой стрелке.
Тоже самое, для  NumPad 7, 9.
Home
в Preview Window вращение объекта
Insert
в Preview Window вращение объекта в другую сторону.
NumPad - ON
если включено, то цифровые клавиши выполняют функции стрелок.
W S A D
в Preview Window объектов передвигает просматриваемый объект.
 
 
LMB
перемещение объектов в одной плоскости
RMB
вращение объектов в одной плоскости.
MMB
плавное передвижение камеры в окне рендера. Держать ролик нажатым и двигать мышь.
LMB 2x
двойной клик - вызов окна свойств «кликнутого» объекта.
LMB 2x slow
двойной клик с увеличенной паузой, дает возможность переименовывать объекты, так и вводить нужное кол-во предметов в инвентарь существ и нпсов.
LMB + S
скалирование размера выделенного объекта с жатой ЛКМ.
(если предварительно в настройках оного поставить 2.0, или 0.5, то изменение размера в окне рендера позволить создать объект 4.0, или 0.25 размера)
LMB + V
зуминг на выбранный объект.
LMB + Z
перемещение объекта по оси Z
LMB + Y
перемещение объекта по оси Y
LMB + X
перемещение объекта по оси X
RMB + Z
вращение объекта по оси Z
RMB + Y
вращение объекта по оси Y
RMB + X
вращение объекта по оси X
MMB + I
Get data about references? Выводит информацию о референсах.
 
 
Shift + LMB
Зажать Shift и не отпуская жать ЛКМ.
Бывает сложно попасть в эту "паузу".
Но если попасть снимает выделение и двигает камеру, если отпустить Shift, будет выделен
первый попавший под мышь объект.
Shift +
движение мыши без зажатых кнопок
вращение камеры в окне рендера, вокруг Выбранного предмета! Обращайте на это внимание.
Shift + ARR
медленное перемещение по сцене, т. е. сцена не улетает за край окна.
Shift + C
скрывает все объекты оставляя видимой только землю! Удобно при редактировании ландшафта.
Shift + Home
вращение сцены вокруг выбранного объекта.
если объекта нет, просто вращение камеры
Shift + PgUp
вращение в другую сторону.
 
 
ALT + F
Быстрый доступ к меню File
ALT + E
меню Edit
ALT + V
меню View
ALT + D
меню World
ALT + C
меню Character
ALT + G
меню Gameplay
ALT + H
меню Help
ALT + PrintScrin
обычный скриншот Render Window.
ALT + F4
логично, закроет редактор.
ALT + F6
переключение между окном рендера и окном объектов, т.е. переводит фокус.
Что позволяет работать с клавиатуры...
 
 
ARR
в Render Window резкий бросок в сторону, т.е. очень быстрое перемещение по сцене.
ARR
в Preview Window вращение объекта
ARR
в меню диалогов перемещают выбранную реплику вверх вниз. ОДНАКО!
Это задевает все прочие реплики в приделах топика, т.е. может привнести в плагин кучу мусора и создать иные конфликты.
Пользуйтесь этой возможностью только для диалогов внутри вашего плагина!
Т.е. в приделах созданных вами топиков!
