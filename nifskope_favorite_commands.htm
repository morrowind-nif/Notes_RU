Nifskope Favorite Commands

Самые ходовые опции!
Для нифскопа 1.1.3 версии (С)
1.
New Window
наипервейшая опция при работе с несколькими нифами и копированием данных между оными.

Если запустить одновременно две копии программы, то копирование между файлами будет не возможно(С)
 
2.
Reset Block Detail
полезна для возвращения к исходному (стартовому состоянию) окна детализации.
 
А также, для быстрой замены текстур, имена которых идут здесь на одном уровне с шейпами и прочим!
 
3.
Show Hidden
Показывает скрытые объекты в сцене.
 
4.
Combine Properties
Удаляет все повторы свойств.
Рекомендуется всегда использовать на финальном этапе работ с моделью!
 
5.
Insert
Позволяет добавить в файл нечто новое из появляющегося по ПКМ списка.
 
Контроллеры, шейпы, ноды, свойства и еще много чего.
По умолчанию, бОльшая часть - для старших Тесов.
Однако это можно исправить путем замены Nif.xml файла.
 
6.
Attach:
 
Attach parent node
Attach effect
Attach properties
Attach node
Весьма полезная опция!
Позволяет быстро присоединить нечто новое к выделенному объекту.
Упаковывает выбранный объект в ноду. Т.е. делает объект вложенным в состав ноды.
Добавляет эффект (текстурный, или вертексного освещения).
Добавляет свойства.
Вставляет в выбранный узел новую новую ноду.
 
7.
copy branch (ctrl+C)
Самая ходовая опция!
Копирует выбранную связь, со всеми ее потрохами и настройками!
Это наипервейший инструмент при обработке Нифов!
 
8.
paste branch (ctrl+v)
логично, следующая по востребованности опция.
Вставляющая скопированное в указанное место.
 
9.
Apply
Устанавливает текущие локальные координаты объектов, как 0.0.0, что помогает избавиться от смещения объекта. Весьма полезно при копипасте объектов с одного место на другое.
 
Именно устанавливает в приделах самого объекта, а не меняет на глобальные!
 
Критически важно при внесении изменений на корневую ноду.
Если не нажать, в игре ничего не измениться.
Т.е. если перемещался корневая нода, то эффект в игре будет только после установки новых координат точки опоры, в нулевые значения.
 
10.
Convert
быстро превращает объект одного типа, в другой.
Ninode ->NiBsAnimationNode например.
 
Активно не везде.
 
11.
Flag
Смена флага позволяет быстро поменять некоторые настройки (свойства) объекта, либо скрыть выбранный объект.
Меню флага доступно не на всех объектах!
Хотя параметр флаг, есть практически у всего.
 
12.
Combine Shapes
Позволяет склеить несколько шейпов с одинаковым набором свойств, в один.
Бывает полезна при завершения процесса оптимизации модели.
 
13.
Update Centr/Radius
Позволяет исправить графическую ошибку отображения модели.
Это весьма важная опция!
см. примечания здесь.
 
14.
Crop to branch
Позволяет удалить из файла, все кроме выделенного объекта.
Что бывает полезно для дальнейшего копирования оного в другой ниф файл, либо для быстрой очистки.
 
15.
remove by IDE
Позволяет быстро вычистить файл от ненужных объектов.
Весьма актуально при чистке Ниф файлов существ.
 
 
Опасные опции:
Remove Bogus Nodes
и
FIX BIP01
 Лучше обходить стороной!
 
 
Fix Bip01 - The latest addition to the spell book magically transforms skeletons imported+exported with blender back into the original Morrowind (n)pc skeleton. If you encounter problems with armor or clothing parts showing up twisted like mad inside Morrowind cast this spell on the Bip01 Node.
Fix Bip01  - это опция исправления скелета экспортированного из Блендера (актуально, только, для старых версий Ниф плагина).
Во всех остальных случаях обходить стороной!
