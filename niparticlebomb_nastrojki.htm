NiParticleBomb настройки

NiParticleBomb
 
Бомба должна находится точно в координатах частиц, или где-то близко, иначе эффекта от нее будет заметно!
 
Параметры.
Параметр бомбы "Chaos" не поддержан в ниф файлах!
 
Для нормального редактирования и создания этого элемента требуется старая версий нифскопа, либо, что предпочтительнее - замена Nif.Xml файла.
Есть в приложении к заметками.
Версия Нифскопа 1.1.3 или 1.2 альфа.
 
Без замены ХМЛ файла, нифскоп 1.1.3 не сможет даже открыть модель содержащую этот элемент.
Попытки сохранить, приведут к повреждению ниф файла!

Описание.
Unknown Float 1
Decay*
= range
Указывается максимальная дальность разлета частиц.
Если Unlimited range, то должно быть пусто.
 
*Вероятно неправильно назван параметр. Это Дистанция а не Decay.
 
Unknown Float 2
Duration
= duration
Продолжительность взрыва, время - во фреймах.
 
Unknown Float 3
DeltaV
= Strength
Сила взрыва.
Может принимать отрицательные значения.
Это будет притягивать частицы, а не выталкивать от центра "взрыва".
 
Unknown Float 4
Start
= Start time
Фрейм начала взрыва
 
Unknown Int 1
Decay Type
Explosion Parametrs
0 Unlimited range, 1 liner, 2 Exponential
Без ограничений, линейный разлет, по экспоненте.
 
После замены Ниф.Хмл файла:
DECAY_NONE
DECAY_LINEAR
DECAY_EXPONENTIAL
 
Unknown Int 2
Symmetry Type
Blast symmetry
0 сфера, 1 цилиндр, 2 планар.
Тип взрыва, сферический, цилиндрический, по плоскости.
 
После замены Ниф.Хмл файла:
SPHERICAL_SYMMETRY
CYLINDRICAL_SYMMETRY
PLANAR_SYMMETRY
 
Unknown Float 5
N\A
X position
Позиции бомбы по осям. Это важно для ее эффекта на частицы!
Особенно в режиме "плоскости".
 
Значения в ниф файле = значением в МАХ
X 15.6482 Y 10.706 Z 8.3853 в МАХ будут без изменений записаны в Ниф файл.
(как кажется (С)
 
Unknown Float 6
N\A
Y position
 
Unknown Float 7
Position
Z position
После замены ХМЛ файла, все значения позиции бомбы сместились в одну строку.
Указывающую позицию сразу по всем осям.
 
Unknown Float 8
N\A
Y rotation
Поворот по осям.
При этом цифры в нифскопе никак не отражают значения в МАХ.
Y 90 в МАХ = -1, -0, -0  в ниф файле.
 
Unknown Float 9
N\A
X rotation
X 90 в МАХ = -0, -1, -0.
 
Unknown Float 10
Direction
Z rotation
После замены ХМЛ файла, все значения поворота бомбы сместились в одну строку. Указывающую поворот сразу по всем осям.
 
Это актуально только для Цилиндра и Плоскости.
Для сферы не будет иметь никакого значения.
 
Верно для старой версии нифскопа:
Если повернуть бомбу на 90* в МАХ здесь значение не изменится.
Но если повернуть ее по осям XY здесь значение будет меняться:
Х 88, Y 55, Z 0 = -0.0286, 0.9994, -0.0200
